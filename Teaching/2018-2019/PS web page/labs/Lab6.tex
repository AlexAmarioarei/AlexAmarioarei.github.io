Spunem cƒÉ o variabilƒÉ aleatoare \(X\) este repartizatƒÉ \emph{normal} sau

\section{Reparti»õia Gama}\label{repartitia-gama}

Spunem cƒÉ o variabilƒÉ aleatoare \(X\) este repartizatƒÉ \emph{Gama} de
parametrii \((\alpha, \beta)\), cu \(\alpha, \beta > 0\), »ôi se noteazƒÉ

\[
